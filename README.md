# 🛡️ Fenrir Security

Fenrir Security is a modern React-based frontend application built with **Vite** and **Tailwind CSS**.  
The project provides a clean authentication flow and security dashboard interface including login, scan, and dashboard pages.

---

## 🚀 Tech Stack

- ⚛️ React (JSX)
- ⚡ Vite
- 🎨 Tailwind CSS
- 📦 ESLint
- 🧩 Lucide Icons
- 📁 Modular Component Structure
